Danetian has nineteen consonant phonemes:
* six plosives
* six fricatives
* two affricates
* five sonorants


| | Labial | Dental | Palatal | Velar |
|:---|:---:|:---:|:---:|:---:|
| Plosive | p, b | t, d | | k, g |
| Affricate | | ts | tʃ, dʒ | |
| Fricative | f, v | s, z | ʃ, ʒ | x |
| Nasal | m | n | | |
| Trill | | r | | |
| Approximant | | l | j | |
