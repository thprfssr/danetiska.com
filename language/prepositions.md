[Prepositions](https://en.wikipedia.org/wiki/Preposition) are used to denote
various spatial, temporal, and other relationships. In Danetian, each
preposition governs one or more
[cases](https://en.wikipedia.org/wiki/Grammatical_case), which affects the
meaning of the preposition.

| Preposition | Case | Meaning |
|:--|
| ana | acc. | onto |
| ana | loc. | on |
| anti | gen. | before (temporal or spatial); in front of |
| apo | abl. | from, away from |
| desj | abl. | without |
| do | gen. | until |
| en | acc. | into |
| en | loc. | in |
| endo | gen. | inside of |
| esj | abl. | out of |
| esjtro | gen. | outside of |
| medi | gen. | between |
| nid | acc. | down |
| opi | loc. | by, beside |
| peri | acc. | around, about (circular motion) |
| peri | gen. | about (topic of discussion) |
| posti | gen. | after (spatial or temporal) |
| pri | gen. | near |
| pro | acc. | for, towards |
| proti | acc. | against |
| rodi | gen. | for the sake of |
| seponts | acc. | according to |
| sjom | ins. | with |
| tros | acc. | through |
| ud | acc. | up, outward |
| uper | loc. | over |
| upo | loc. | under |
